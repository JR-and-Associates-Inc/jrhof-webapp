# Contributing

Thank you for helping preserve the history of Colorado high school baseball umpiring. Useful contributions include factual corrections backed by sources, accessible website fixes, documentation improvements, and carefully identified historical material.

Open the matching structured issue before substantial work. Keep pull requests focused, explain the evidence for content changes, preserve historical routes and redirects, and follow the checklist in the pull-request template. Run `npm ci`, `npm run check`, `npm run build`, `npm run validate`, and `git diff --check`.

Do not submit secrets, personal information, private account identifiers, unlicensed media, live payment tests, or unapproved claims about prices, tax treatment, events, people, or nonprofit benefits. Security vulnerabilities must use GitHub private vulnerability reporting, not a public issue or pull request.

This public repository has no open-source license. Public visibility does not grant permission to reuse the code, name, branding, photographs, or historical material. By contributing, you confirm you have the right to submit the work for JR and Associates, Inc. to review and use; acceptance is not guaranteed.

Maintainers may request sources, accessibility changes, smaller commits, or removal of sensitive material. All participants must follow the Code of Conduct.
