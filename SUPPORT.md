# Support

This repository is maintained for the Joe Rossi Umpires Hall of Fame public website. It does not provide emergency, legal, tax, payment, or account support.

- Website bugs, public content corrections, historical-material offers, and non-sensitive hardening proposals: use the matching structured GitHub issue.
- General Hall of Fame, event, donation, sponsorship, or records questions: use [jrhof.org/contact](https://jrhof.org/contact/).
- Security vulnerabilities: use [GitHub private vulnerability reporting](https://github.com/JR-and-Associates-Inc/jrhof-webapp/security/advisories/new) or the private contact in `SECURITY.md`. Never open a public vulnerability issue.

Do not include attendee data, dietary notes, payment identifiers, credentials, private documents, or other personal information. Volunteer response times vary; submitting a request does not guarantee a particular resolution or deadline.
