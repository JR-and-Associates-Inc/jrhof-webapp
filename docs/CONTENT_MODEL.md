# Inductee Content Model

This is an active generated-data contract for the Astro site. Treat `src/data/inductees.json` as generated output, not as a file to hand-edit. The source inputs are the original CSV, DOCX biographies, photo candidates, and reconciliation artifacts listed below.

## Location

The candidate dataset is `src/data/inductees.json`. It is generated by `scripts/generate_inductee_candidates.py` from the archived source/reconciliation package moved during the 2026-06-29 repo cleanup:

- `_archive/repo-cleanup-2026-06-29/content/All Inductees.csv`
- `_archive/repo-cleanup-2026-06-29/_migration/source-reconciliation/updated-inductee-source-status.csv`
- `_archive/repo-cleanup-2026-06-29/_migration/reconciliation/inductee-reconciliation.json`
- readable files under `_archive/repo-cleanup-2026-06-29/content/Bios/`
- verified-candidate files under `_archive/repo-cleanup-2026-06-29/content/Photos/`

The root-level `content/` and `_migration/` directories are no longer part of the active Astro project tree. Restore or update the archived source package deliberately before regenerating public data.

## Record fields

Each of the 150 records contains:

- `stable_id`
- `display_name`, `sort_name`, and `aliases`
- `induction_year` or `induction_era`
- `canonical_slug`
- `current_wordpress_url`
- `proposed_canonical_url`
- `legacy_repo_url` and `redirect_sources`
- `bio_status`, `bio_source`, and extracted `biography` paragraphs
- `portrait_status`, `portrait_source`, `portrait_output_filename`, and `portrait_url`
- `live_link_status`
- `migration_lane`
- `publication_status`
- `board_review_required`
- `reviewer_notes`
- `source_provenance`

## Publication behavior

- `bio_status=available`: render extracted original-source paragraphs.
- `bio_status=pending_review`: render “Biography pending review.”
- `portrait_status=verified_candidate`: render the copied person-specific source photo.
- `portrait_status=pending_review`: render `/images/inductees/portrait-pending.svg`.
- `board_review_required=true`: show a review notice without exposing internal notes as asserted facts.

## Regeneration

Run `npm run content:generate`, then `python3 scripts/generate_redirects.py`. Review the diff before committing because source approvals can change publication status and routes.
